To finalize the installation and get started with sensenet ECM Notification, please follow these steps:

(install guide is accessible online at: http://community.sensenet.com/docs/install-notification-from-nuget)

1. Build your solution, make sure that there are no build errors.

2. Install sensenet ECM Notification using SnAdmin

   - open a command line and go to the \Admin\bin folder
   - execute the install-notification command with the SnAdmin tool

   .\snadmin install-notification   

You are good to go! Hit F5 and start registering notifications!
For more information and support, please visit http://sensenet.com
